Chapter - 01

We start building a website by creating a file named index.html.

index.html is a special filename which is presented when the website root address is typed in the browser.

A Basic HTML Page

<!DOCTYPE html>
<html>
<head>
<title>Bright Future</title>
</head>
<body>
<h1>This is a heading</h1>
<p>My paragraph</p>
</body>
</html>

Explanation of Tags

  • <!DOCTYPE html> → Specifies this is an HTML5 document.
  • <html> → Root of the HTML page.
  • <head> → Contains page metadata.
  • <title> → Shows the page title in the browser tab.
  • <body> → The visible content of the page.
  • <h1> → Heading tag.
  • <p> → Paragraph tag.

What is an HTML Tag?

A tag works like a container for content or other HTML elements.

HTML Document → Browser → Rendered Web Page

Important Notes

  • Head and Body tags are children of the HTML tag.
  • HTML is the parent of Head and Body tags.
  • Most HTML elements have an opening tag and a closing tag.
  • The content is written between the opening and closing tags.
Tip: HTML tags act like containers that hold and structure the content of a webpage.
