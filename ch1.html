HTML Beginner Guide

Understanding the basic structure of websites

Chapter 0 – Introduction

HTML stands for Hyper Text Markup Language. It is the basic language used to create web pages.

HTML defines the structure of a webpage using elements called tags. These tags tell the browser how content should appear.

Why HTML, CSS and JavaScript?

Modern websites are built using three main technologies.

  • HTML – Defines the structure of the webpage (the backbone).
  • CSS – Adds styling like colors, layout, fonts and design.
  • JavaScript – Adds functionality and interactivity.

Example: Showing or hiding text when a user clicks a button.

Example of a Simple HTML Page

This is a basic HTML structure:

<!DOCTYPE html>
<html>
<head>
<title>My First Page</title>
</head>
<body>
<h1>Hello World</h1>
<p>My first website</p>
</body>
</html>

Installing VS Code

To write HTML we use a text editor. One of the most popular editors is Visual Studio Code.

  • Go to Google
  • Search for VS Code
  • Download and install it

Although HTML can also be written in Notepad, editors like VS Code make coding easier.

Types of Websites

There are mainly two types of websites:

  • Static Website – Content stays the same for every user.
  • Dynamic Website – Content changes depending on user interaction or database.

Popular Code Editors

  • VS Code
  • Atom
  • Sublime Text

Hosting a Website

After creating a website, it must be hosted online so people can access it. Hosting platforms store your website files and make them available on the internet.

Example hosting platforms:

  • GitHub Pages
  • Netlify
  • 300webhost
